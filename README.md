# PracticeDSA

Collection of Data Structures and Algorithms practice problems implemented in Python.  
This repository is used for interview preparation and strengthening problem-solving skills.

## Topics Covered

- Arrays
- Strings
- Sliding Window
- Stack & Queue
- Linked List
- Sorting Algorithms
- Binary Search
- Concurrency (Producer Consumer)
- Miscellaneous Problems

## Sample Problems

- LRU Cache
- Min Stack
- Longest Substring Without Repeating Characters
- 3 Sum
- Merge Sort
- Sliding Window Maximum
- Binary Search Variations

## Goal

Consistent DSA practice and interview readiness.

## Language
Python
